NUMA

  • Non-Uniform Memory Access 의 약자, 멀티프로세서 환경에서 적용되는 메모리 접근 방식
  • CPU 마다 메모리를 보유하고 있는데, CPU 가 자신의 메모리에 접근하는 것을 Local Access라고 부른다.
  • 노드 : CPU와 그에 해당하는 메모리를 합쳐서 부르는 말
  • NUMA에서 다른 노드에 접근하는 것을 Remote Access 라고 부른다.
