    ffmpeg
  • local file (mp4 -> gif)

local file (mp4 -> gif)

ffmpeg \
  -i <input-filename> \
  -r 15 \
  -vf scale=512:-1 \
  -ss 00:00:03 -to 00:00:06 \
  <output-filename>

  • 간단한 옵션

  • -r : frame rate

  • -ss : start offset
