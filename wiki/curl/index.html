파일 다운

curl -o <filename> <url>

curl -O <url>

For REST API

  • -L : follow redirect, -v : verbose, -H : Header

Basic

curl -L <url> | jq .

With Auth

curl -v -L -H 'Authorization: Bearer <token>' <url>

Change method

curl -v -L -X POST <url>

File

curl -T <filename> <url> # method 랑 조합해서 사용할것

Form

curl -d "foo=bar&x=y" <url> # 자동으로 POST 로 처리
